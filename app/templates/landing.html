Cómo crear un usuario de solo lectura

Recomendado para mayor seguridad — aunque puedes usar tu usuario habitual.

  1. Ve a Ajustes → Usuarios y crea un usuario nuevo.
  2. Asígnale el perfil Portal o un perfil personalizado sin derechos de edición.
  3. Activa acceso de lectura a Ventas, Inventario, Contabilidad y CRM.
  4. Establece una contraseña y anótala para el formulario.
  5. Opcional pero recomendado: usa una API Key en vez de contraseña — en Ajustes → Usuario → Seguridad.
